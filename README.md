## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Nazarkdh/NewsGrid.git
   cd NewsGrid
   ```

2. Open the project in your browser:
   - Simply open `index.html` in your web browser
   - Or use a local development server for better experience

3. To set up a local server (optional):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

Then navigate to `http://localhost:8000` in your browser.

## Customization

### Colors & Styling
Edit the CSS files in the `css/` directory to customize colors, fonts, and layout.

### Images
Replace images in the `img/` directory with your own news article images.

### Content
Edit the HTML files to update article content, titles, and descriptions.
